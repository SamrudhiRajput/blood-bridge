About Blood Bridge

Blood Bridge is a college project that demonstrates a sleek, professional front-end for a community-driven blood donation system. It includes donor registration, request submission, a searchable donors directory, and instant certificate generation for donors.

How to use

  1. Register as a donor on the "Become Donor" page.
  2. If you need blood, submit a request on the "Request Blood" page.
  3. View registered donors on the "Available Donors" page.
